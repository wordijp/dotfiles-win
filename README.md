# dotfiles-win
Windows dotfiles.

## Installation

ファイル群のインストール
```
> make install
```

コマンドプロンプトのAutoRun設定
```
> set_cmd_autorun.reg
```


## Requirement

下記ツールを利用しています

- [wordijp/mycommands(GitHub)](https://github.com/wordijp/mycommands)
- [Make for Windows](http://gnuwin32.sourceforge.net/packages/make.htm)
